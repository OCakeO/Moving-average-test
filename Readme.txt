Если Вы это читаете, значит, вы собираетесь стать частью 
команды КБ Медицинских изделий и микроскопии АО "ПО "УОМЗ".
Мы с нетерпением ждем Вас!

В данном задании необходимо разработать модуль для фильтрации
ряда целых чисел методом скользящего среднего на языке С.
API модуля находится в "app/include/MA_Filter.h". Для выполнения
задания необходимо дополнить исходный код в "app/src/MA_Filter.c".

Критерием работоспособности модуля является прохождение тестов
в "test/MA_Filter_test.cpp", написанных с использованием фреймфорка
CppUTest (https://cpputest.github.io/). Вы также можете воспользоваться
этими тестами для самопроверки. Для этого (на платформе Windows):
	* Установите CygWin https://www.cygwin.com/install.html.
	* При установке, дойдя до этапа Select Packages, выберете View = Category,
	и напротив пакета Devel выберете Install, чтобы установить набор средств, необходимых для разработки.
	* Откройте Cygwin и перейдите в директорию /test с помощью команды cd.
	* Введите команду "make". В результате будет выполнена сборка Вашего исходного
	кода и кода тестов, а также будут выполнены тесты. Пример выполнения тестов:
		!!!!!!!.
		OK (8 tests, 1 ran, 0 checks, 7 ignored, 0 filtered out, 2 ms)

Вначале все тесты, кроме одного, отключены. Чтобы включить тест сотрите IGNORE_ в
IGNORE_TEST(GroupName, TestCaseName). Название теста TestCaseName отражает суть 
проверки. Тесты отсортированы по сложности и глубине покрытия модуля. Рекомендуется
открывать по одному новому тесту по мере прохождения предыдущих.

В случае возникновения вопросов или комментариев, обращайтесь к выдавшему задание.
